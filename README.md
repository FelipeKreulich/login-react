## Como Rodar Este Projeto? - ⚙️ <div id="#rodar"></div>

``` bash
# Clone este repositório
$ git clone https://github.com/FelipeKreulich/login-react

# Acesse a pasta do projeto em seu terminal
$ cd login-react

# Instale as dependências da Aplicação
$ npm install

# Execute a aplicação
$ npm start
```
